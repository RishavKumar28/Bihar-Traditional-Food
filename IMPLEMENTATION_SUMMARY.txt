## 📊 How It Works

### Customer Flow:
```
1. Customer places order
   ↓
2. Sees order success page with "Track Order" button
   ↓
3. Clicks button → goes to order-tracking page
   ↓
4. Sees timeline starting at "Order Placed" (PENDING)
   ↓
5. Admin changes status → Page detects change in 10 seconds
   ↓
6. Customer sees notification and timeline updates
   ↓
7. Status progresses: Pending → Processing → Delivered
```

### Admin Flow:
```
1. Admin goes to Orders management page
   ↓
2. For each order, there's a status dropdown
   ↓
3. Admin selects new status (e.g., "Processing")
   ↓
4. Form auto-submits
   ↓
5. Database updates with new status and timestamp
   ↓
6. Customer's page automatically shows update
```
